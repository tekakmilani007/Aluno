# Git Comandos

-git clone (clona um repositório github para um local)
-git status (mostra o status do repositório local)
-git add nome-arquivo (adicionar o arquivo no stage changes)
-git add . (adicionar todos os arquivos do path no stage changes)
-git checkout -b (cria uma nova branch)
-git commit -m (salva os arquivos no repositórioi local)
-git push (enviar os dados do repositório local para o repositório remoto)
